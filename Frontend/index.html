How ratings work (chess-style)

  • Each player starts around 1200 rating.
  • Win vs higher-rated player → bigger rating gain.
  • Win vs similar rating → moderate rating gain.
  • Win vs lower-rated player → small rating boost.
  • Lose vs lower-rated player → bigger rating loss.

Behind the scenes, an Elo-like formula updates rating based on opponent strength, quiz difficulty, and final score—similar to chess.com rating behaviour.

Mode-based ratings

  • Quiz (Rapid) – standard class quizzes.
  • Tournament (Blitz) – fast-paced event games.
  • Practice (Casual) – practice mode, rating optional.

Just like chess.com has Rapid / Blitz / Bullet, Samarpan can track separate ratings per mode, plus a global rating for ranking.

About Samarpan

AI-powered quiz portal by Team Samarpan – designed as an upgrade over Kahoot-style platforms with automation, analytics, and rating.

Project overview

Samarpan is an AI Quiz Portal built for the Smart Education theme. It auto-generates quizzes from text/PDFs, supports multiplayer modes, and tracks ratings, ranks, and performance over time.

  • AI quiz generation from study material.
  • Live scoring, leaderboards and tournaments.
  • Dynamic rating & ranking system with XP and levels.

Tech stack

  • Frontend: HTML, CSS, JS single-page UI.
  • Backend: Node.js + Express REST API.
  • Realtime: Socket.io for live rooms & scores (future).
  • AI Service: Python microservice with OpenAI / NLP models.
  • Database: MongoDB Atlas for users, quizzes & leaderboards.
